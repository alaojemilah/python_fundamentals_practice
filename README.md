## Notebook Structure
The notebook is organized as a series of dated practice sessions and assignment questions, each with the problem statement followed by the working solution. Examples include:

- Multiplication table generator using a for loop and range
- Sum of numbers from 1 to N using a for loop
- Vowel and consonant counter using string iteration
- Factorial calculator using a for loop with edge-case handling
- Duplicate removal from a list while preserving order
- Sorting planet names by distance from the sun (case normalization and ordered matching)
- Shipping cost calculator processing a batch of orders with conditional logic
- Battery drain simulator using a while loop
- Currency converter function (USD to NGN)
- Discount calculator function
- Finding numbers divisible by 7 but not 5 within a range
- Fibonacci sequence using recursion
- Random greeting generator using the random module
